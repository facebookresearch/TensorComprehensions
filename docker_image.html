Installing TC from docker image¶

We provide docker runtime images for conda and non-conda environments both. TC officially supports running gcc 4.*, CUDA 8, CUDNN 6 and ubuntu14.04 and gcc 5.*, CUDA9, CUDNN6 on ubuntu16.04

The conda and non-conda images for each setup are below:

  • Ubuntu 14.04 conda environment
$ docker run -i -t tensorcomprehensions/linux-trusty-gcc4.8-tapir5.0-cuda8-cudnn6-py3-conda:x86
  • Ubuntu 14.04 non-conda environment
$ docker run -i -t tensorcomprehensions/linux-trusty-gcc4.8-tapir5.0-cuda8-cudnn6-py3:x86
  • Ubuntu 16.04 conda environment
$ docker run -i -t tensorcomprehensions/linux-xenial-gcc5-tapir5.0-cuda9-cudnn7-py3-conda:x86
  • Ubuntu 16.04 non-conda environment
$ docker run -i -t tensorcomprehensions/linux-xenial-gcc5-tapir5.0-cuda9-cudnn7-py3:x86

TC runtime image with nvidia-docker¶

We also provide a runtime nvidia-docker image for Ubuntu 14.04, gcc 4.8, CUDA 8 and CUDNN 6. Using this image, you can also run gpu tests. To run the image, make sure you have nvidia-docker installed. Then run the image using following command:

  • NVIDIA-Docker Ubuntu 14.04 conda environment
$ nvidia-docker run --rm -i -t tensorcomprehensions/trusty-gcc4.8-py3-conda-cuda:x86
